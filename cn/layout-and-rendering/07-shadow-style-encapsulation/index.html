测试用例7：Shadow DOM 样式封装隔离

统一风格的多页测试，覆盖基础隔离、slot 分发、:host/::slotted 选择器以及多实例性能观察。

Chromium 最新版

测试页面列表

01 基础隔离
验证外部 CSS 不影响 Shadow 内部；内部样式不外泄。
 02 slot 分发与 ::slotted
验证插槽内容继承外部样式，并可通过 ::slotted 定制样式。
 03 :host 选择器
使用 :host、:host(...) 等控制宿主外观与状态样式。
 04 多实例与样式隔离
创建大量组件实例，观察样式计算/渲染是否相互干扰。

使用说明

  • 所有页面顶部均提供工具栏，可切换“全局干扰样式”与可视化轮廓。
  • 各页面的自定义元素会在构造函数中执行 attachShadow({ mode: 'open' }) 并注入 Shadow 层级结构与内部 <style>。
  • 请配合 Chrome DevTools 展开组件，查看 Shadow Root 与样式作用域。
