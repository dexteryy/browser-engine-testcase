测试用例7：Shadow DOM 样式封装隔离 — 基础隔离

验证 Shadow 内部样式不受页面全局 CSS 影响，且不会影响外部。

Chromium 最新版

测试区域

外部普通文本（类名 inside，应为蓝色）

上方组件中段落应显示为红色，表示 Shadow 内部样式生效且未被外部蓝色样式覆盖。

测试目标

确认 Shadow DOM 的样式封装：外部 CSS 无法选中 Shadow 内部节点；内部样式也不会泄漏到 Light DOM。

方案与使用

  • 页面通过 global.css 定义 .inside 与 my-element p 的蓝色样式，模拟“全局干扰”。
  • <my-element> 在构造函数中 attachShadow({ mode: 'open' }) 并在 Shadow 内注入 <style> 设为红色。
  • 使用上方工具栏开关可动态启用/停用外部样式或内部样式，观察隔离效果。

期望结果

  • “外部普通文本”应为蓝色（受全局样式影响）。
  • 组件内部段落应始终为红色（仅受 Shadow 内部样式控制）。
  • 关闭“启用全局干扰样式”后，外部文本应回退为默认色；组件内部不受影响。
