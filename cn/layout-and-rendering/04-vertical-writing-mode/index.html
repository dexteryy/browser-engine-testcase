测试用例4：垂直书写模式与文本方向

验证 vertical-rl / vertical-lr 下中英文混排、字符旋转与布局定位

说明

本测试用例包含多个独立页面，每个页面专注于一种测试场景。所有页面采用统一的白底样式与操作栏，方便观察、对比与回归测试。

页面索引

  • 01 基础：vertical-rl 与水平对照
  • 02 字符方向：mixed / upright / sideways 对比
  • 03 布局整合：垂直容器与 Flex / Grid 对齐
  • 04 标点与数字：竖排下的位置、合字与括号

建议使用最新的 Chromium 进行测试。

如何使用

  • 进入各页面后，通过工具栏切换 writing-mode、text-orientation 等参数，观察文本方向、字符旋转与换行表现。
  • 开启“基线栅格”和“调试轮廓”辅助观察行距和盒模型边界。
  • 在布局整合页面中，尝试改变 Flex/Grid 的对齐参数，查看不同书写模式的尺寸计算与对齐关系。
