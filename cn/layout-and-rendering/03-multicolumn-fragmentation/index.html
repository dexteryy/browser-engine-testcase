测试用例3：多列布局（Multicolumn）与内容分栏

用于评估 Chrome 对 CSS 多栏布局（columns）的渲染正确性与性能。

本目录包含多个独立页面，每个页面针对一个细分测试目标。请按需打开：

  • 固定列数：column-count
  • 固定列宽：column-width（自动列数）
  • 复杂分栏与避断：Fragmentation 压力与性能
  • 跨栏元素：column-span: all
  • 书写方向与列顺序：writing-mode / direction

UI 风格统一：白底、顶栏通栏、其他区域统一宽度。页面文字均为中文，代码注释为英文。

说明

  • 每个页面顶部提供工具栏控件，可动态调整列数、列间距、列分隔线等参数。
  • 测试区域展示真实的多栏内容，包含段落、小标题与图片等。
  • 建议在 DevTools 的 Performance 面板记录首次渲染与滚动性能；如需，可启用 Paint Flashing 辅助观察重绘。
